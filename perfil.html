Descrição dos projetos:

Projeto 1: Site de Verificação de Idade- Nele você seleciona sua idade e clica o botão ao lado onde irá mostrar uma imagem de uma pessoa que aparenta ter essa idade (Está dividida em Criança, Adolescente, adulto e idoso), juntamente a sua idade selecionada e uma mensagem

Projeto 2: Site de Verificação das horas- Neste site você tem a escolha de colocar a hora desejada e aparecerá uma mensagem junto a hora escolhida mas também tem um botão onde dirá exatamente a hora, minutos, segundos e milisegundos.

Projeto 3: Site flex- Neste projeto está demosntrando a diversidade das posiçôes e dimensôes atraves de div's com posicionamentos do tipo: Sticky, Estática, Relativa, Fixa e Absoluta.

Projeto 4: Site com a hora do dia- Neste projeto queriamos aperfeiçoar novas funçôes em JavaScript pois a cada hora do dia, que estará no horário atual, irá mudar a mensagem junto com as imagens da paisagem que são de montanhas com o céu mostrando como estaria naquela hora do dia, e também o background (fundo do site) mudará de cor.

Projeto 5: Calculadora- Neste último projeto simulamos uma calculadora da Web onde aprimoramos as funçôes em JavaScript e os cálculos que programamos para que saiam os resultados em baixo e de forma correta para o funcionamento ideal da calculadora.
